# Инструкция по настройке проекта

## Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

После установки выполните команду для проверки совместимости версий:

```bash
npx expo install --fix
```

Это автоматически обновит все Expo-зависимости до версий, совместимых с SDK 54.

### 2. Настройка API URL

API сервер уже настроен и доступен по адресу: `http://158.160.9.21:5000/api`

Если необходимо изменить URL API, откройте файл `src/config/api.js` и измените значение `API_BASE_URL`.

### 3. Запуск приложения

```bash
npx expo start --clear
```
