# Лингвистический анализ текстов

Мобильное приложение для полного цикла лингвистического анализа научных текстов на русском языке.

## Описание

Приложение выполняет следующие виды анализа текстов:
- **Частотный анализ** - подсчет частоты слов
- **Терминологический указатель** - автоматическое формирование списка терминов
- **Именной указатель** - поиск и категоризация имен собственных
- **Конкорданс** - поиск контекстов для терминов
- **Словарь определений** - создание словарных статей с определениями

## Технологии

- React Native
- Expo SDK 54
- AsyncStorage для локального хранения данных
- Полностью офлайн (без сервера)

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

3. Откройте приложение в Expo Go на вашем устройстве:
   - Скачайте Expo Go из App Store или Google Play
   - Отсканируйте QR-код из терминала

## Функциональность

### 1. Загрузка текстов
- Загрузка .txt файлов
- Ручной ввод текста
- Указание категории текста (информатика, лингвистика, медицина)
- Сохранение текстов в локальную базу данных

### 2. Частотный анализ
- Подсчет частоты слов
- Сортировка по убыванию частоты
- Визуализация частоты

### 3. Терминологический указатель
- Автоматическое формирование списка терминов (минимум 100)
- Поддержка словосочетаний (2-3 слова)
- Выделение аббревиатур
- Возможность ручного редактирования

### 4. Именной указатель
- Поиск имен собственных
- Категоризация: персоналии, топонимы, организации, программные продукты
- Фильтрация по категориям

### 5. Конкорданс
- Поиск контекстов для терминов
- Минимум 3 контекста для одиночных слов
- Минимум 1-2 контекста для словосочетаний

### 6. Словарь определений
- Создание словарных статей
- Определения терминов
- Связанные словосочетания
- Примеры употребления из конкорданса

## Дизайн

Приложение использует темную тему с минималистичным дизайном:
- Темный фон (#0a0a0a)
- Акцентный цвет: индиго (#6366f1)
- Outlined-кнопки
- Карточки для контента

## Особенности

- Полностью офлайн работа
- Локальное хранение всех данных
- Работа только с русскоязычными текстами
- Поддержка формата .txt
- Все вычисления выполняются на устройстве
